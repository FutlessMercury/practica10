¿Qué es CSS?

Significa "Cascading Style Sheet". Las hojas de estilo en cascada se utilizan para formatear el diseño de las páginas web . Se pueden usar para definir estilos de texto, tamaños de tablas y otros aspectos de las páginas web que anteriormente solo se podían definir en el HTML de una página .

1 ¿Cómo funciona CSS?

Como lenguaje de estilo, CSS especifica cómo los usuarios ven los documentos, desde el diseño hasta el estilo. Los documentos afectados suelen ser archivos de texto que se estructuran a partir de un lenguaje de marcado, el más común de los cuales es HTML; sin embargo, XML y SVG también son populares. Para presentar el documento al usuario, CSS lo convierte del archivo de texto en una forma utilizable. Con los navegadores, esta presentación ocurre en una pantalla de computadora, impresora o proyector.


Una vez que haya cargado y analizado el contenido HTML y el estilo CSS, el proceso para combinarlos ocurre en dos etapas separadas. Primero, el navegador los convierte en el modelo de objeto de documento (DOM). Una vez que el DOM combina el contenido y el estilo del documento y es una representación del documento dentro de la memoria de la computadora, el navegador mostrará el contenido.

2 Ventajas
  • Mejor velocidad del sitio web
  • Sitios webs más fácil de mantener
  • Permite un diseño consistente
  • Ahorro de tiempo de desarrollo
  • Mejor compatibilidad para dispositivos
3 Desventajas
  • Genera confusión debido a muchos niveles de CSS
  • Problemas entre navegadores
  • Problemas de seguridad
  • Implica trabajo extra para desarrolladores
